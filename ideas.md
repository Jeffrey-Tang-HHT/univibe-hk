# UniVibe HK — Revised Design Direction

## Selected: "Clean Neon Minimal"

A simple, modern dark-themed landing page. No HK street photography. Instead, use abstract geometric shapes, subtle gradient backgrounds, and clean typography to communicate trust, exclusivity, and youthful energy.

### Core Principles
1. **Simplicity First** — Clean layouts, generous whitespace, no visual clutter
2. **Dark + Neon Accents** — Deep navy/midnight background with coral & cyan accent pops
3. **Typography-Driven** — Let bold headlines and clear copy do the heavy lifting
4. **Geometric Motifs** — Small abstract shapes (hexagons, circles, triangles) as subtle decorations referencing the anonymous avatar system

### Color Palette
- Background: Deep Navy (#0A0E27) to Midnight (#050816)
- Primary Accent: Electric Coral (#FF6B6B)
- Secondary Accent: Neon Cyan (#00F5D4)
- Tertiary: Soft Lavender (#C4B5FD)
- Text: Off-white (#E8E6F0), muted (#8B8CA0)

### Typography
- Display: "Space Grotesk" — geometric, techy
- Body: "DM Sans" — clean, readable
- Mono: "JetBrains Mono" — for stats/numbers

### Layout
- Clean single-column flow with centered content
- Feature sections use simple icon + text cards in grids
- No heavy imagery — rely on icons (Lucide), subtle gradients, and CSS shapes
- Smooth scroll-triggered fade-in animations
